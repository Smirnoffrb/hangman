Игра "Виселица"

Игра написана на языке Ruby

____

Правила игры
-----------

Игроку необходимо угадать загаданное слово, поочередно вводя буквы.
Количество возможных ошибок 7.

Запуск программы
-----------

```
ruby main.rb
```

Добавление новых слов
----------
Новые слова можно добавить в файл /data/words.txt
